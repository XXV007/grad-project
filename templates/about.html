About This Project

Project Overview

This Multimodal and Robust Deepfake Detection System is a graduate research project developed for CPSC 589 at California State University Fullerton.

The system combines advanced machine learning techniques to detect manipulated videos by analyzing both spatial features (frame-level anomalies) and temporal patterns (motion inconsistencies across frames).

Technical Architecture

  • Spatial Analysis: EfficientNet/XceptionNet CNN for frame-level feature extraction
  • Temporal Analysis: LSTM/3D-CNN for motion pattern recognition
  • Fusion Strategy: Late fusion combining spatial and temporal features
  • Explainability: Grad-CAM and temporal activation visualizations
  • Backend: Flask, PyTorch, OpenCV
  • Frontend: Bootstrap 5, JavaScript

Datasets

The system is trained and evaluated on multiple benchmark datasets:

  • FaceForensics++: High-quality face manipulation dataset
  • DFDC: Deepfake Detection Challenge dataset
  • Celeb-DF: Celebrity deepfake dataset
  • ForgeryNet: Multi-method forgery dataset
